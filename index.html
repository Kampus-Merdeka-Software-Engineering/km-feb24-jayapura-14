About The Project

Project ini bertujuan untuk memenuhi tugas akhir MSIB Studi Independen RevoU TechAcademy. Pada project akhir ini, team 14 Jayapura menggunakan keterampilan analisis data mengenai data penjualan Giant Superstore dan pembuatan web yang telah dipelajari selama berjalannya program ini. Adapun keterampilan analisis data yang dilakukan oleh team 14 Jayapura yaitu, melakukan eksplorasi data, identifikasi tren penjualan, analisis profitabilitas produk, visualisasi data, dan memberikan rekomendasi untuk meningkatkan kinerja penjualan. Hasil analisis tersebut kemudian digunakan untuk merancang dan membangun sebuah website yang informatif dan fungsional. Project ini tidak hanya mengasah kemampuan teknis dalam analisis data dan pengembangan web, tetapi juga memperkuat kemampuan team 14 Jayapura dalam mengkomunikasikan informasi secara efektif melalui media digital.
